# 🌌 Futuristic Developer Portfolio

A modern futuristic portfolio website built to showcase skills, projects, and GitHub analytics with a clean neon-inspired UI.  
Designed for developers who want a sleek and professional online presence.

## ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/bhardwajyuvika2305/bhardwajyuvika2305.github.io-Personal-Portfolio.git
```

Open the project folder:

```bash
cd bhardwajyuvika2305.github.io-Personal-Portfolio
```

Install dependencies:

```bash
npm install
```

Run the project:

```bash
npm run dev
```
